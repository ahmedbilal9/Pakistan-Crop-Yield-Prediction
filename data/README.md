# Dataset Documentation

## Overview
This directory contains the agricultural datasets for Pakistan crop yield prediction.

## Directory Structure

### `raw/`
Contains the original, unprocessed datasets.

**Expected File:**
- `crop_yield_data.csv` - Pakistan agricultural records dataset

**Dataset Source:**
- FAOSTAT (Food and Agriculture Organization Corporate Statistical Database)
- Contains agricultural data from 1961 to 2023

## Dataset Description

### Features
The dataset includes the following information:

| Column | Description | Type |
|--------|-------------|------|
| Year | Year of agricultural record | Integer |
| Item | Type of crop (e.g., Wheat, Rice, Cotton) | Categorical |
| Value | Yield value (output per hectare) | Numerical |
| Unit | Unit of measurement | Categorical |
| Flag | Data quality flag | Categorical |

### Original Columns (Before Preprocessing)
- Domain Code, Domain
- Area, Area Code (M49)
- Element Code, Element
- Item Code (CPC), Item
- Year Code, Year
- Unit
- Value
- Flag Description, Flag
- Note

### Preprocessing Steps
1. **Column Filtering**: Removed redundant and non-unique columns
   - Dropped: Domain Code, Domain, Area, Area Code, Note, Year Code, Element Code, Item Code, Flag Description
   
2. **Element Filtering**: Filtered to only include "Yield" records
   
3. **Missing Values**: Handled missing values by removing incomplete records
   
4. **Encoding**: One-hot encoded categorical variables (Item, Unit, Flag)

### `processed/`
Will contain preprocessed datasets ready for model training (generated by the pipeline).
